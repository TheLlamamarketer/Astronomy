# Astronomy

This repository contains weekly astronomy tasks and projects.

## Folder Structure

Each task is organized in its own folder:

- **Task1/** - Week 1 materials and assignments
- **Task2/** - Week 2 materials and assignments
- **Task3/** - Week 3 materials and assignments
- **Task4/** - Week 4 materials and assignments
- **Task5/** - Week 5 materials and assignments

## Adding New Tasks

To add a new task for a new week, create a new folder following the naming convention `TaskN` where N is the task number. Include a README.md in each task folder describing the specific assignment or project.
